
# Прогнозирование оттока клиентов (Churn Prediction) — сравнение моделей машинного обучения

## 📌 Описание задачи
Клиенты телеком-компании уходят к конкурентам, что приводит к потере выручки. Цель проекта — разработать модель машинного обучения, которая по историческим данным (демография клиента, подключённые услуги) предскажет вероятность оттока для каждого клиента. Это позволит бизнесу своевременно предлагать удерживающие мероприятия и сократить потери.

## 📊 Данные
*   **Источник:** Kaggle (dataset по оттоку клиентов)
*   **Размер:** 5282 строки, 20 признаков (числовые и категориальные).
*   **Целевая переменная:** `Churn` (1 — клиент ушёл, 0 — остался).

## 🔍 Что сделано
*   **Разведочный анализ (EDA):** Визуализация распределений, выявление выбросов и анализ взаимосвязей признаков с целевой переменной.
*   **Предобработка данных:** Обработка пропусков, кодирование категориальных признаков (OneHotEncoding), масштабирование числовых данных.
*   **Feature Engineering:** Создание новых признаков, улучшающих качество модели.
*   **Моделирование:** Реализован пайплайн для обучения и сравнения 4 моделей классификации: логистическая регрессия, метод опорных векторов (SVM), случайный лес (Random Forest) и градиентный бустинг (XGBoost). Проведён подбор гиперпараметров с помощью GridSearchCV.

## 📈 Результаты
| Модель | Accuracy | ROC-AUC |
| :--- | :--- | :--- |
| Логистическая регрессия | 0.77 | **0.85** |
| SVM | 0.80 | 0.84 |
| Random Forest | 0.80 | 0.83 |
| XGBoost | 0.79 | 0.84 |

**Лучшая модель:** Логистическая регрессия, показавшая наилучший компромисс между точностью и полнотой.

## 💡 Выводы и бизнес-ценность
Разработанная модель позволяет с высокой точностью выявлять клиентов, склонных к оттоку. Это даёт бизнесу возможность:
*   Применять точечные удерживающие стратегии (персональные скидки, специальные предложения).
*   Оптимизировать маркетинговый бюджет, не тратя его на лояльных клиентов.
*   Потенциально снизить отток на **10-15%** и повысить LTV (пожизненную ценность) клиента.
